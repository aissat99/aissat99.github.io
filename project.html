[Algorithms] LAN simplified

An application of the Dijkstra algorithm with shortest path search within a network of pseudo servers.
The goal is to mimic a real-world request-response process: a client tries to request a pseudo page and makes a request to the closest server hosting the website on the network. The server then sends the requested page back to the client.
